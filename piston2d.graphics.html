piston2d.graphics module¶

Graphics module mirroring piston2d_graphics for Python built in Rust.

Note

This does not mirror all functions, but is in active developement. It is, however, compulsory to use this module as the current implementation of the binding does not allow the specific creation of any of the shape objects (e.g. Rectangle)
